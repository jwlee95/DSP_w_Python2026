## 소개

디지털 신호처리를 처음 접하는 학부생을 위해, 이론적 유도 과정을 파이썬 코드로 직접 확인할 수 있도록 만든 실습 자료입니다. 이산신호의 기초 개념부터 샘플링 이론, 푸리에 해석, z변환과 디지털 필터 설계까지 다루며, 각 이론은 심전도(ECG), 뇌파(EEG), 근전도(EMG)와 같은 실제 생체신호에 적용해 보는 것으로 마무리됩니다.

- **저자**: 이정환 (JeongWhan Lee)
- **소속**: 과학기술대학 ICT융합공학부 바이오메디컬공학과
- **대상 과목**: 생체신호처리 (2026년도 2학기, 학부 3학년)

## 실행 환경 (Requirements)

본 저장소의 모든 코드는 다음 환경에서 실행 및 검증되었습니다.

- Python 3.10 이상
- `numpy`
- `scipy`
- `matplotlib`

```bash
git clone https://github.com/jwlee95/DSP_w_Python2026.git
cd DSP_w_Python2026
pip install -r requirements.txt
```

각 장 폴더로 이동하여 스크립트를 직접 실행할 수 있습니다.

```bash
cd ch01_signals_and_systems
python ch01_example.py
```

모든 스크립트는 별도의 데이터 다운로드 없이 **코드 자체만으로 합성 신호를 생성**하도록 작성되어 있으므로, 인터넷 연결 없이도 동일한 결과가 재현됩니다.
